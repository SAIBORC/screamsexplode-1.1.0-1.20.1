# ScreamsExplode

Scream into your mic and explode in Minecraft! A standalone Fabric 1.20.1 mod with direct mic capture, configurable sensitivity, and multiplayer support.

## Requirements

- Minecraft 1.20.1
- Fabric Loader 0.15+
- Fabric API 0.92.9+ (required)
- Mod Menu 7.0.0+ (optional, for config screen)

## Build

```bash
./gradlew clean build --no-daemon
```

Output: `build/libs/screamsexplode-1.1.0-1.20.1.jar`

## Usage

1. Place the JAR in your `mods/` folder (both client and server)
2. Launch the game
3. Scream into your microphone
4. You explode

Configure sensitivity via Mod Menu or edit `config/screamsexplode.json`.
